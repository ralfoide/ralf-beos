virtual void ProcessMessage(ui32 msg, ui32 lparam, vptr data, ui32 *channel=NULL)
Process one msg in the main app loop. Note that the inherent flaw in the lack of allocation policy for the data block.
Parameters:
msg - the msg id to be processed
lparam - a long value associated with the message
data - a pointer onto a data block
channel - a pointer onto a socket channel id, used for networking communication

Process one system request to open a document
