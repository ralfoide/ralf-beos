enum EFileMode

Open File Mode for RIBin

Documentation

Open File Mode for RIBin. Maps the standard fnctl access modes.
Parameters:
kFileReadOnly - Read only file.
kFileWriteOnly - Write only file.
kFileReadWrite - Read and write file.
kFileInvalidMode - Not to be used, represents the non initialized state of the RIBin, used for file not already open or already closed.
