Documentation

Interface for manipulating a seekable stream device. Currently only used by RBinFile. Should be used by RBinMemory too.
virtual rbool Open(const sptr path, EFileMode mode)
Opens a file for reading or writing
Returns:
true on success, false on failure to locate or open the file.
Parameters:
path - The ASICZ string path to the file.
mode - An open mode.

virtual rbool Open(const RPath &path, EFileMode mode)
Opens a file for reading or writing
Returns:
true on success, false on failure to locate or open the file.
Parameters:
path - The RPath path to the file.
mode - An open mode.

virtual void Close(void)
Closes a previously open file. Must not fail if the file was not open.

virtual void Rewind(void)
Rewinds an open file (i.e. seeks to the beginning)

virtual rbool Seek(si32 position, EFileSeek mode = kFileSeekSet)
Seeks to a given position in the file
Returns:
true if seeked successfully, false otherwise.
Parameters:
position - The position to seek, in bytes.
mode - The relative or absolute seek reference mode.

virtual si32 Tell(void)
Reports the current position in the file stream
Returns:
The number of bytes from the beginning of the file.

virtual rbool Eof(void)
Reports if the end of while was reached. Eof is always reached on an invalid file.
Returns:
true if the current position is the end of file. false otherwise.

virtual si32 Size(void)
Reports the size of the file. Gets the file size using seek/tell (valid for read-only).
Returns:
The size of the file, in bytes.

virtual rbool Read(char *buffer, si32 size)
Reads a bunch of bytes
Returns:
true if the requested number of bytes was correctly read, false if the file was not open, or nothing or less than the amount of requested data was actually read.
Parameters:
buffer - The pointer onto the buffer to put the read data. Must be preallocated onto a storage big enough to accomodate 'size' bytes; non-NULL pointer.
size - The number of bytes to be read from the stream.

virtual rbool Write(char *buffer, si32 size)
Writes a bunch of bytes
Returns:
true if the requested number of bytes was correctly written, false if the file was not open, or nothing or less than the amount of requested data was actually written.
Parameters:
buffer - The pointer onto the buffer to get the written data. Must be preallocated onto a storage big enough holding at least 'size' bytes; non-NULL pointer.
size - The number of bytes to be written to the stream.
